# A1. Слияние

> Два состояния можно объединить через merge, mergetool и commit

> Участвуют три стороны: current, incoming и base

> Идентичные изменения в разных ветках не приводят к конфликтам

-  — объединить текущую ветку с другим коммитом
-  — объединить текущую ветку с другим коммитом, причем обязательно создать новый объединяющий коммит
- No files need merging — разрешить имеющиеся конфликты
-  — отменить слияние
